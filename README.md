## 使用说明

### 1. 配置EmailJS
1. 访问 [EmailJS官网](https://www.emailjs.com/) 注册账号
2. 创建Service和Template
3. 在配置页面填入相关信息
4. 测试连接确保配置正确

### 2. 客户上传流程
1. 选择活动类型和套餐
2. 填写完整活动信息
3. 点击"SUBMIT FOR REVIEW"提交
4. 等待Admin审核结果通知

### 3. Admin审核流程
1. 使用默认密码登录Admin面板
2. 查看待审核的上传请求
3. 批准或拒绝申请
4. 导出已批准数据

### 4. 生成Events页面
1. 在编辑器中完善活动信息
2. 点击"生成Events页面"
3. 系统自动生成events.html文件
